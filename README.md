## Идея

**Одним предложением:** Физическая реальность — это "тень" многомерного информационного пространства.

```
         χ-ПРОСТРАНСТВО (информация)          ФИЗИЧЕСКИЙ МИР (то что видим)
         ═══════════════════════════          ════════════════════════════
         
              Поле Φ(χ)          ────────►    Волновая функция ψ(x)
                  │                                    │
                  │              Проекция              │
                  ▼                                    ▼
           Метрика g_μν(χ)       ────────►    Гравитация, ОТО
                  │                                    │
                  ▼                                    ▼
         Близость в χ            ────────►    Квантовая запутанность
                  │                                    │
                  ▼                                    ▼
         χ-канал передачи        ────────►    Квантовая телепортация
```

**Конечная цель:** Понять механизм квантовой нелокальности настолько глубоко, чтобы приблизить реальную телепортацию состояний.

## Дисклеймер

Это **гипотеза**, не доказанная теория. Математически согласованная модель с работающими симуляциями, но без peer review и экспериментального подтверждения. Цель — исследование идеи.

## Эксперименты

### 1. Ядро проекции K(x, χ) — ФУНДАМЕНТ

```bash
python chi_projection_kernel.py
```

Вывод ядра проекции из вариационного принципа. Показывает: принцип неопределённости, суперпозиция, запутанность и коллапс — всё следует из формы K.

### 2. Базовое χ-пространство

```bash
python chi_space_theory.py
```

Создание χ-пространства, запутанных пар, визуализация: частицы далеки в xyz, но совпадают в χ.

### 3. Квантовая механика из χ

```bash
python qm_from_chi_theory.py
```

Суперпозиция, интерференция, коллапс — всё из проекции χ → xyz.

### 4. Гравитация из χ

```bash
python gravity_from_chi.py
```

Закон Ньютона φ = −GM/r из уравнения Пуассона в χ-пространстве.

### 5. Телепортация через χ-поле

```bash
python experiments/nonlocal_psi_teleportation.py
```

Попытка передачи состояния через общее χ-поле запутанных частиц.

## Теория

### Ядро проекции (выведено из вариационного принципа)

```
K(x, χ) = N · exp(-(x - π(χ))² / 4σ²) · exp(iφ(χ)·x/ℏ)

σ² = ℏ / (2mω_χ)     ← ширина определяется динамикой χ
N  = (2πσ²)^(-1/4)   ← нормировка
```

### Действие

```
S = ∫ dⁿχ √|g| [ (c⁴/16πG_χ) R + L_Φ ]
```

### Лагранжиан χ-поля

```
L_Φ = (iℏ/2)(Φ*∂τΦ - Φ∂τΦ*) 
    - (ℏ²/2m)|∇χΦ|² 
    - V(χ)|Φ|² 
    - (λ/4)|Φ|⁴
    - κ|Φ|²log|Φ|²    ← информационный член
```

### Что следует из одного действия

| Физика | Как получается |
|--------|----------------|
| Квантовая механика | Проекция Φ(χ) → ψ(x) |
| Общая относительность | Проекция g_μν(χ) → гравитация |
| Квантовая гравитация | Суперпозиция геометрий в χ |
| Запутанность | Общие χ-координаты |
| Телепортация | Передача Φ(χ) через χ-канал |
| Тёмная материя | χ-структуры без ЭМ-проекции |
